# MiniBook

MiniBook est un réseau social étudiant minimaliste conçu en BUT Informatique pour un projet fonctionnant entièrement dans le navigateur, sans serveur ni base de données.

## Fonctionnalités

- Inscription et connexion avec photo de profil
- Modification des informations du compte
- Publication de messages avec ou sans image
- Fil d'actualité avec les publications de tous les utilisateurs
- Recherche de publications par texte et par auteur
- Système de likes et suppression de ses propres publications
- Persistance des données via le localStorage

## Pages

| Page | Description |
|------|-------------|
| Accueil | Page de bienvenue avec accès à la connexion et l'inscription |
| Inscription | Création d'un compte avec photo de profil |
| Connexion | Authentification avec email et mot de passe |
| Profil | Affichage des informations de l'utilisateur connecté |
| Modification | Mise à jour des informations du compte |
| Fil | Fil d'actualité avec publication rapide et recherche intégrée |
| Publication | Création d'une publication avec image |
| Recherche | Résultats de recherche filtrés par texte et auteur |

## Technologies

- HTML / CSS / JavaScript 
- localStorage pour la persistance des données
- Google Fonts (Montserrat, Poppins)

## Lancement

Aucune installation requise. Il suffit d'ouvrir le fichier `index.html` dans un navigateur.
